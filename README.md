## サービス概要
釣具専門の架空のECサイト。

## コンセプト
TSURINCHUは、沖縄で釣りを楽しんでいる釣りんちゅ（釣り人）にもっと快適に、もっと釣果を上げてもらいたいと思いで誕生した、オンラインショップです。沖縄の独特なフィールドに合わせたロッドやリール、様々な魚種に対応したルアー、暑い日が多い沖縄だからこそ、快適に過ごせるためのウェアなど沖縄の釣りに特化したブランドです。
